# TryHackMe – CTF notes

This repository documents my **hands-on learning, notes, and write-ups** while practicing cybersecurity on **TryHackMe**.  
The goal of this repo is to **track progress, reinforce concepts, and build a strong practical foundation** in offensive and defensive security.

> ⚠️ **Disclaimer**  
> All content in this repository is for **educational purposes only**.  
> No real-world systems are targeted.  
> Flags, credentials, and sensitive information are **never shared**.

TryHackMe is a hands-on cybersecurity learning platform that provides guided labs and real-world simulations covering:
